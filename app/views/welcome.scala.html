API REST for Open Source projects

You can use the following endpoints to obtain and create open source projects and its releases

  • GET
    /projects

  • GET
    /projects/$id

  • GET
    /projects/find?name=$name

  • POST
    /projects Body = { "name" : "examplename" }

  • GET
    /projects/$projectId/releases

  • GET
    /projects/releases/$id

  • POST
    /project/releases Body = { "version": "0.0.2", "link": "http://myrepo.org/version=isa002" }
